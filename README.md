# game website vue
My project to learn vue, and typescript

## Functionality
- change theme
- tic-tac-toe
- coin flip
- memory cards

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
